# It's my amazing project of  git explanation  
---

 * to initiate git repository  

 
```bash
git init
```

* to add new files


```bash
git add .
```


* to commit changes 

```bash
git commit -m "text of comment"
```

* to link remote repository


```bash
git remote add origin
```


* to push repository to remote


```bash
git push -u origin
```
